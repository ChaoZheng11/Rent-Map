# React+Express+MongoDB实现租房地图
    目前demo还在改进中
    城市定位目前锁定为南京
    搜索功能以及租房详情还在调整中
# 技术栈
* React.js
* Ant Design
* React-router
* Redux
* Axios
* Express
* puppeteer
* Mongoose
    
# 运行
    cd backend 
    npm i
    modify Mongodb connection link in ./db/db.js
    npm run start 
    
    cd rent
    npm i
    npm run start
